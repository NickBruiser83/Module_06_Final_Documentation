Description of the purpose of the site

The purpose of scaredyet.com is to display the top horror movies in a few subcategories. The user will navigate the site using the menu to access the voting category or the dropdowns for either the top 10 for Sci/fi or Drama horror. When users select one of the top 10 options, they will be brought to a page with ten links ranked 1 - 10 of the best horror movies in that subgenre. The links take the user to youtube, where they can watch the trailer for the film. The last option in the main menu is the vote button. The vote button will take the user to a page with several entry fields to enter their name, number, email, and favorite movie. After submitting the details, the user is taken to a page showing all the favorites saved to their local computer.

Descriptions of buttons on each page, the user actions, and the outcomes or results when they are clicked.

There are 4 buttons that link to more pages on the website in the menu of every page.

-The first button says Home and is Linked to the Main page.

-The second button says Sci/fi Horror and when the mouse hovers over it a dropdown menu is displayed. The links in the dropdown menu are top-10, top-100, and upcoming. The top 10 has the only link that works and goes to a top 10 page.

-The third button says Drama Horror and when the mouse hovers over it a dropdown menu is displayed. The links in the dropdown menu are top-10, top-100, and upcoming. The top 10 has the only link that works and goes to a top 10 page.

-The fourth button says vote and links to a page with a submission form.

On the Sci/fi Horror page, there is a ranked list with 10 Sci/fi horror movies. Each movie has a link that takes the user to the film's trailer on youtube.

On the Drama Horror page, there is a ranked list with 10 Drama horror movies. Each movie has a link that takes the user to the film's trailer on youtube.

On the voting page are five entry fields. The user can enter their first name, last name, email, phone number, and favorite horror movie.

underneath the entry fields on the vote page is two buttons.

-the first button is labeled Clear local storage and, when pressed, will clear the local storage on the user's computer

-The second button is labeled submit and will save what is entered in the entry fields to local storage. After the submit button is pressed and the entries are held, the user is directed to a page that lists all the favorite movies saved on their computer's local storage.

On the top left of the header on the voting page is a panel labeled More voting sites. When the panel is pressed, the user is taken to a page with fake links to more voting sites.
